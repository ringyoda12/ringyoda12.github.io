# 개인정보 처리 안내 (PubMed 연구 GPT)

본 GPT는 식품 개발 연구소 내부 연구 지원을 목적으로 제작되었습니다.

## 1. 외부 API 사용
본 GPT는 논문 검색을 위해 NCBI PubMed E-utilities API(https://eutils.ncbi.nlm.nih.gov/)를 사용합니다.

사용자가 입력한 검색어(예: 원료명, 기능성 키워드 등)는 논문 검색을 위해 해당 API 서버로 전송될 수 있습니다.

## 2. 수집 및 저장 정보
- 본 GPT는 사용자의 개인정보를 수집하거나 저장하지 않습니다.
- 입력된 검색어는 논문 조회 목적에 한하여 일시적으로 외부 API로 전달됩니다.
- 별도의 데이터베이스에 저장·보관·판매하지 않습니다.

## 3. 이용 목적
- PubMed 논문 검색 및 연구 결과 요약
- 식품 개발 연구 의사결정 지원

## 4. 문의
본 GPT 관련 문의는 연구소 내부 담당자에게 문의하시기 바랍니다.
